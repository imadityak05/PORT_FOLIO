1. Clone the repository:
   `ash
   git clone https://github.com/imadityak05/PORT_FOLIO.git
   cd portfolio
   `

2. Install dependencies:
   `ash
   npm install
   # or
   yarn install
   `

3. Start the development server:
   `ash
   npm run dev
